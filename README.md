##  Scientific Calculator with Plotting:
- A powerful GUI-based scientific calculator built using Python and Tkinter, featuring advanced mathematical operations and function plotting.

## Features:
- Basic arithmetic operations (+, −, ×, ÷)
- Scientific functions (sin, cos, tan, log, ln, √)
- Constants (π, e)
- Binary ↔ Decimal conversion
- Previous answer (ANS) support
- Function plotting with custom range
- Interactive GUI with modern design

##  Technologies Used
- Python
- Tkinter (GUI)
- Matplotlib (Plotting)
- NumPy

##  Function Plotting
- Plot up to 2 functions simultaneously
- Custom X-axis range
- Supports expressions like:
   - sin(x)
   - cos(x)
   - log(x)
